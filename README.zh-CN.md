# Improved Collapsed Tabs

[English](readme.md) | [中文](README.zh-CN.md)

这些修改只会在浏览器侧边栏折叠时生效。

功能：
 - 让当前标签页更明显：在侧边显示指示条
 - 移除音频播放图标
 - 让正在播放声音的标签页更明显：在侧边显示指示条
 - 可选隐藏标签页的关闭按钮
 - 可配置标签页标题显示的字符数量，并可选择显示在上方或下方

设置项包括：
 - 仅强调当前选中标签页的标题
 - 在固定标签页和 essential tabs 中也显示标题
 - 在侧边栏折叠模式下自动隐藏工具栏
 - 设置触发工具栏浮动的辅助按键
 - 隐藏标签页的关闭按钮
 - 设置标签页关闭按钮大小
 - 设置标签页关闭按钮位置
 - 设置标签页标题宽度
 - 设置标签页标题高度
 - 设置标签页标题字体大小
 - 设置标签页标题字体
 - 设置标签页标题圆角
 - 将标签页标题显示在下方
 - 设置指示条颜色
 - 设置指示条宽度

注意：工具栏浮动辅助按键需要一个很小的 JS 脚本。如果你通过 GitHub 直接用 Sine 安装，需要开启 `sine.allow-unsafe-js`，或从 Sine store 安装此模块，脚本才会被加载。
